NATO Translator - Progressive Web App

A simple Progressive Web Application that allows you to translate the characters in a string in to the NATO phonetic alphabet.
Ever wanted to spell out a word like they do in the military movies? Now you can with this tiny web app that works offline! Made with Polymer 2.0. Source
